# Data Dictionary — hotel-pvar-drivers-linreg

The data is generated by `src/pvar_linreg/dgp.py`. The schema below is what
`pvar_linreg.data.load()` returns.

## Columns

| Column | Type | Definition |
| ------ | ---- | ---------- |
| `property_id` | int | unique synthetic property identifier |
| `property_tier` | categorical | "Strategic" / "Preferred" / "Standard" / "Long-tail"; mix 10/20/40/30% |
| `country` | categorical | top-12 APAC + Europe + "OTHER" |
| `star_rating` | float | 2.0 .. 5.0 |
| `contracted_base_rate` | float | derived from tier + country + star (USD) |
| `date` | date | observation day |
| `day_of_week` | int | 0..6 |
| `month` | int | 1..12 |
| `is_weekend` | bool | day_of_week ∈ {5, 6} |
| `is_holiday` | bool | probabilistic via a holiday calendar |
| `season_multiplier` | float | sinusoidal annual + holiday spikes |
| `lead_time_days` | int | days until arrival for the rate snapshot |
| `channel` | categorical | "Direct" / "ChannelManagerA" / "ChannelManagerB" |
| `channel_noise` | float | per-channel gaussian noise on rate |
| `has_rate_parity_issue` | bool | hidden ground-truth label; depends on tier × channel |
| `supplier_quality_score` | float | per-supplier latent quality |
| `posted_rate` | float | rate Agoda would display |
| `expected_rate` | float | contractually-expected rate |
| `PVar` | float | `(posted_rate - expected_rate) / expected_rate` |
| **`PVar_abs`** | float | `abs(PVar)` — primary regression target |

## Engineered features

| Feature | Definition |
| ------- | ---------- |
| `log_lead_time` | `log1p(lead_time_days)` |
| `month_sin`, `month_cos` | cyclical encoding |
| `log_expected_rate` | controls for absolute price level |
| `star_rating_centered` | `star_rating - 3.0` |

Interactions (added by the OLS formula):

- `C(property_tier):C(channel)` — the substantively interesting one.

## True coefficients

`pvar_linreg.dgp.true_coefficients()` returns the dict of values embedded in the DGP.
The appendix uses this to prove that the regression recovers them within bootstrap CI.
